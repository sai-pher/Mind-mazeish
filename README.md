# Mind Maze

[![CI — Analyze & Test](https://github.com/sai-pher/mind-mazeish/actions/workflows/ci.yml/badge.svg)](https://github.com/sai-pher/mind-mazeish/actions/workflows/ci.yml)
[![CD — Build & Publish APK](https://github.com/sai-pher/mind-mazeish/actions/workflows/cd.yml/badge.svg)](https://github.com/sai-pher/mind-mazeish/actions/workflows/cd.yml)

A medieval castle trivia game for Android, inspired by Microsoft Encarta's Mind Maze. Navigate 10 castle rooms, answer Wikipedia-grounded trivia questions generated by Claude, and read the source articles in-app.

---

## Download

The latest test APK is attached to the [**Latest CI Build**](https://github.com/sai-pher/mind-mazeish/releases/tag/latest-build) pre-release — updated automatically on every push to `main`.

> Signed with the debug keystore. Enable **Settings → Apps → Install unknown apps** on your device before installing.

---

## Setup

```bash
# 1. Clone
git clone https://github.com/sai-pher/mind-mazeish.git
cd mind-mazeish

# 2. Add your Anthropic API key
echo "ANTHROPIC_API_KEY=sk-ant-..." > .env

# 3. Install dependencies
flutter pub get

# 4. Run on a connected device or emulator
flutter run
```

## Architecture

| Layer | Technology |
|-------|------------|
| State management | Riverpod (`NotifierProvider`, `AsyncNotifierProvider`) |
| Navigation | GoRouter |
| Question pipeline | Wikipedia REST API → Claude API (`claude-sonnet-4-20250514`) |
| Article viewer | `webview_flutter` (Wikipedia mobile) |
| Animations | `flutter_animate` |
| Fonts | Cinzel, Cinzel Decorative, Lora (Google Fonts) |

## Screens

| Route | Screen |
|-------|--------|
| `/` | Start Screen — animated title, "Enter the Castle" |
| `/game` | Gameplay — question fetch, answer buttons, fun fact sheet |
| `/article` | Article Viewer — in-app Wikipedia WebView |
| `/results` | Results — score, accuracy, star rating, restart |

## CI / CD

| Workflow | Trigger | What it does |
|----------|---------|--------------|
| **CI** | Push (non-main), PR → main | `flutter analyze --fatal-infos` + `flutter test` |
| **CD** | Push to `main`, manual | Analyze → Test → Build release APK → Publish to GitHub Releases |

Add `ANTHROPIC_API_KEY` as a repository secret to bake the key into CI builds.
